# Real Estate Investment Analyzer

A powerful real estate investment analysis tool designed to help investors quickly evaluate rental properties, estimate cash flow, calculate returns, and identify profitable investment opportunities.

## Features

### Property Analysis
- Purchase price analysis
- Market value comparison
- After Repair Value (ARV) calculations
- Rent estimate analysis
- Property tax and insurance estimates
- HOA fee tracking
- Vacancy and maintenance forecasting

### Investment Metrics
- Cap Rate
- Cash-on-Cash Return
- Net Operating Income (NOI)
- Debt Service Coverage Ratio (DSCR)
- Gross Rent Multiplier (GRM)
- Loan-to-Value (LTV)
- Monthly and annual cash flow
- Total Return on Investment (ROI)

### Financing Analysis
- Conventional loan calculations
- Adjustable down payment scenarios
- Interest rate comparison
- Amortization schedules
- Mortgage insurance calculations
- Closing cost estimates

### Expense Tracking
- Property taxes
- Insurance
- Property management
- Maintenance reserves
- Capital expenditure (CapEx) reserves
- Utilities
- HOA dues
- Vacancy allowance
- Miscellaneous operating expenses

### Portfolio Management
- Analyze multiple properties
- Compare investment opportunities
- Save favorite listings
- Export reports
- Historical property tracking

### Data Sources
- MLS listings
- Public property records
- User-provided property information
- Rental market estimates
- Custom financial assumptions
